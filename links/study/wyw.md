# 文言文阅读

### 答题口诀

+ 速读原文，大致划分
+ 标注第三题（文意分析题）
+ 断句要整句
+ 常识题：积累不成看语境
+ 翻译题：注意提示，关注上下句

### 断句技巧

- 找到节点
- 主干完整，主语确定
- 完整成分不能拆
- 排二定一



### 翻译方法

+ 音节——用横线隔开各成分（五字以上）

+ 分辨——不译成分，专有名词保留（人名，地名，官职，年号）

+ 组词——单音节词组词，根据前后词

+ 造句——加入适当的关联词和虚词连词成句，符合现代汉语习惯

+ 猜——利用语境猜意思

+ #### 技巧

+ > + 不认识的字猜偏旁
  > + 两字指向一致(即意思相同)只翻一个
  > + 能用成语则用成语
  > + 积累文件中的310常用字

### 句意技巧

> 不找不同/没有，找能否推翻题目说法。
